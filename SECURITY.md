# Security Policy

## Supported Versions

We are committed to ensuring the security of our users. Currently, we support the following versions of GitLocalize:

| Version | Supported          |
| ------- | ------------------ |
| Cloud   | :white_check_mark: |
| Self-Hosted | :white_check_mark: |

## Reporting a Vulnerability

We take the security of GitLocalize seriously. If you believe you have found a security vulnerability, please **do not open a public issue**.

Instead, please report it to us directly:

1.  **Email**: Send the details to [contact@gitlocalize.com](mailto:contact@gitlocalize.com).
2.  **Details**: Please include as much information as possible (steps to reproduce, affected versions, etc.) to help us verify and fix the issue.

### Our Response

*   We will acknowledge your report within **48 hours**.
*   We will provide an estimated timeline for the fix.
*   We will notify you once the fix is deployed.

Thank you for helping keep GitLocalize safe! ❤️
